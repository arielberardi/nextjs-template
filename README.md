# NextJS Project Template

Ready to use NextJS template for faster development.

## Requirements

* Node >= 22.20

## Features

### Utilities
- React hook form: flexible and extensible forms with validation.
- React Redux Toolkit: global status managment.
- Tanstack Query: data fetching and Server State utilities.
- Zod: runtime type checking.
- Immer: utility for mutations of data.
- Next Intl: internationalization

### Styling
- AntD: React UI library.
- TailwindCSS and Tailwind Animations.
- Clsx: conditional merge of classes.

### Automatic Validations
- Biome for Linter and Formatter.
- Husky for pre-commit checks.
- Commitlint for commit message validation.
